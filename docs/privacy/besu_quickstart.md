## Besu Quickstart

For a quick start with Web3j privacy features on Besu, use the current privacy quickstart recommended in the [Besu documentation](https://besu.hyperledger.org/).

You will need:

- Docker
- Docker Compose or the Docker Compose plugin
- enough memory for multiple local services

Once the quickstart network is running, point Web3j at the exposed Besu JSON-RPC endpoints and use the Besu transaction manager classes shown in the privacy examples.

## Ethereum testnets

The main public Ethereum test networks you are most likely to use with Web3j today are:

- Sepolia
- Holesky

For new development, prefer Sepolia unless you have a specific reason to use another network. It is widely supported by wallets, providers, and tooling.

Holesky is useful when you want a larger validator-style test environment.

For test Ether, use the faucet or provider documentation associated with the network you are targeting.

For local development, it is often simpler to use:

- a local Geth or Besu dev network
- [Web3j EVM](../web3j_evm.md)
- [Web3j Unit](../web3j_unit.md)
